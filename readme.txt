Создание первого проекта:
- Создайте папку для проекта
- Откройте её в VS Code
- В терминале выполните:

dotnet new console

Oтлично! Вы создали новый консольный проект. Теперь вот что делать дальше:

1. Откройте созданные файлы
Проект создал несколько файлов:

- Program.cs - основной файл с кодом
- YourProjectName.csproj - файл проекта

2. Изучите структуру файла Program.cs
Откройте Program.cs - там уже есть базовый код:

// See https://aka.ms/new-console-template for more information
Console.WriteLine("Hello, World!");

3. Запустите проект
В терминале VS Code выполните:

dotnet run

Вы увидите вывод: Hello, World!
